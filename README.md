#  Spotify Data Analysis (SQL Project)

##  Project Overview

This project analyzes a Spotify dataset using SQL to uncover insights about artist performance, country trends, and streaming behavior.

The goal is to demonstrate SQL skills such as JOIN, GROUP BY, HAVING, ORDER BY, and LIMIT while applying real data analysis thinking.

---

The project uses two tables:

## spotify

* song
* artist_id
* streams

## artists

* artist_id
* artist_name
* country

---

#  Tools Used

* SQL (MySQL)

---

##  Business Questions Solved

1. Top artist by total streams
2. Top country by total streams
3. Top 3 artists by total streams
4. Artists with low performance (streams < 3000)
5. Average streams per country
6. Country with highest number of songs
7. Artists from a specific country with high streams
